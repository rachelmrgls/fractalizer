Home

  • P - Pause/play the animation and music.
  • I - Download a screenshot of the current frame.
  • Left/Right Arrow keys - Zoom (fixed step) in X direction.
  • Up/Down Arrow keys - Zoom (fixed step) in Y direction.
  • Left/Right Carats - Zoom (diminishing) in Z direction.
  • D/F - Adjust the real coordinate (value 1) of the Julia set.
  • J/K - Adjust the imaginary coordinate (value 2) of the Julia set. Note that this will adjust the 4th dimension for the 3D Julia.
  • Enter - Swap to the next set of default values for the Julia set.
